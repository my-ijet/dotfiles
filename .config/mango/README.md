# mango-config


# dependence
```bash
yay -S kitty xdg-desktop-portal-wlr swaybg wl-clip-persist cliphist wl-clipboard sway-audio-idle-inhibit-git   brightnessctl grim slurp satty matugen-bin noctalia-shell
```

# Usage
```bash
git clone -b noctalia https://github.com/DreamMaoMao/mango-config.git ~/.config/mango
cp ~/.config/mango/noctalia -r ~/.config/
```
## Some Common Default Keybindings

- super+return: open terminal
- alt+space: open launcher
- super+q: kill client
- super+h/j/k/l: focus direction
- super+m: quit mango
